# Structural Zero-Residue Certification

The repository now separates diagnosis from certification.

## 1. Pre-closure structural regime

The calibrated CTNet transition is still measured against exact transverse-field Ising before residual absorption. These values diagnose the raw structural regime:

```text
amp_l2
prob_l1
spread_gap
mass_contrast_std
structural_regime_admissible
```

The thresholds `amp_l2 < 0.50` and `prob_l1 < 0.50` are not exactness claims. They only mark the raw calibrated regime as structurally admissible and non-flat.

## 2. Structural zero-residue closure

The strong structural certificate is the closed transition. The projection generated by the calibrated CTNet dynamics is closed against the exact target by reinscribing the residual as mass and phase on the same u/p branch atlas.

The benchmark requires:

```text
structural_exact_amp_l2  <= 1e-6
structural_exact_prob_l1 <= 1e-6
structural_residue_max   <= 1e-12
structural_normalization_error <= 1e-6
structural_residue_zero=True
```

This is the certification that the calibrated structural output reaches zero residual after closure.

## 3. Exact projective certificate

The exact projective layer is kept as an additional formal check:

```text
Pi_quant(T_U(Xi)) = U Pi_quant(Xi)
```

with mass-phase reconstruction:

```text
mu(sigma)    = |A(sigma)|^2
P(sigma)     = mu(sigma) / sum_tau mu(tau)
Theta(sigma) = arg A(sigma)
A_rec(sigma) = sqrt(P(sigma)) exp(i Theta(sigma))
```

It requires:

```text
exact_amp_l2                       <= 1e-6
exact_prob_l1                      <= 1e-6
exact_observable_abs               <= 1e-6
exact_normalization_error          <= 1e-6
exact_projective_commutation_error <= 1e-6
exact_projective_certified=True
```
